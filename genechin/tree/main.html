Start looking at the Family Tree by clicking on any of the four indices in the first row above. The 英文目錄 English Index is sorted in the order of the English alphabet. The 中文目錄 Chinese Index is sorted according to the 康熙字典 Kangxi Dictionary (1716 C.E.) order using radicals and stroke counts. The 拼音目錄 Pinyin Index is sorted by using Microsoft Excel. The 台山拼音目錄 Hoisanva Index is sorted by using Microsoft Excel with minor changes to group the same characters together. All four indices are arranged with the Chin (Chan, Chen, Chinn, Chun and other variations in romanization) members first, followed by relatives whose surnames are not Chin. The main portion displays our family tree. Enjoy!
