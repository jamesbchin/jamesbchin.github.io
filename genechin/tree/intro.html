Instructions
Start looking at the Family Tree by clicking on any of the four indices in the first row above. The 英文目錄 English Index is sorted in the order of the English alphabet. The 中文目錄 Chinese Index is sorted according to the 康熙字典 Kangxi Dictionary (1716 C.E.) order using radicals and stroke counts. The 拼音目錄 Pinyin Index is sorted by using Microsoft Excel. The 台山拼音目錄 Hoisanva Index is sorted by using Microsoft Excel with minor changes to group the same characters together. All four indices are arranged with the Chin (Chan, Chen, Chinn, Chun, Tan, Tran and other variations in romanization) members first, followed by relatives whose surnames are not Chin. The main portion displays our family tree. Enjoy!

Note 4 - Meaning of First Column

The first column in all the charts indicates the generation number. The first number is counting from Huangdi. The second number is counting from Mangong. The third number is counting from Shigong. The fourth number is counting from Mo. The generation name is also given starting with the 13th generation from our Great Ancestor Mo [traditional Chinese: 謨; simplified Chinese: 谟; pinyin: Mó; Hoisanva: Mũ]. When a person has a generation name, the personal name is shown in brackets. Generation poems were established for several branches of our family starting from the 13th generation.
